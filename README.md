## Running the app
1. Clone the repository
2. Install Nest CLI
```bash
$ npm install -g @nestjs/cli
```
3. Installation

```bash
$ npm install
```
4. Execute
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
5. Lift database from docker container
```bash
docker-compose up -d
```
